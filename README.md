# ArtTheta Symmetry Tool

## Description

A mini Python-based symmetry drawing tool inspired by mandala art. It uses geometric transformations and trigonometric calculations to generate Vertical, Rotational, and Radial Symmetry.

## Motivation

I wanted to create a project that combines art and programming. Mandala patterns inspired the idea, and the project helped me explore how symmetry can be created using mathematics.

## Features

- Vertical Symmetry
- Rotational Symmetry
- Radial Symmetry
- Mouse-based drawing
- Color selection

## Mathematical Background

This project uses geometric transformations to create symmetrical patterns.

### Vertical Symmetry

Points are reflected across a vertical line.

### Rotational Symmetry

Points are rotated around the center of the screen.

### Radial Symmetry

Points are both reflected and rotated to create repeating patterns.

## Concepts Used

- Reflection
- Rotation
- Coordinate Geometry
- Trigonometry (sin, cos, atan2)
- Distance Formula
